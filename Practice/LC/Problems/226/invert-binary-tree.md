# Table of Contents

# Question

Given the `root` of a binary tree, invert the tree, and return its _root_.

## Example 1

<div align="center" width="100%">
  <img title="Example 1" src="img/example-1.png" />
</div>

### Input

```
root = [4,2,7,1,3,6,9]
```

### Output

```
[4,7,2,9,6,3,1]
```

## Example 2

<div align="center" width="100%">
  <img title="Example 2" src="img/example-2.png" />
</div>

### Input

```
root = [2,1,3]
```

### Output

```
[2,3,1]
```

## Example 3

### Input

```
root = []
```

### Output

```

```

## Constraints

- The number of nodes in the tree is in the range `[0, 100]`.
- `-100 <= Node.val <= 100`
