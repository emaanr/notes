# `Math`

- Exploring some useful methods of the `Math` class in Java.

## `max()`

- Takes two parameters and returns whichever is greater.

## `min()`

- Takes to parameters and returns whichever is lesser.

## `abs()`

- Returns the absolute value of the parameter.

## `sqrt()`

- Returns the square root of the parameter.

## `round()`

- Rounds a variable normally:
  - $4$ or below, let it go.
  - $5$ or above, give it a shove.

## `ceil()`

- Rounds a variable up.

## `floor()`

- Rounds a variable down.
