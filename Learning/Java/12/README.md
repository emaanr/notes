# `while`

- Executes a block of code as long as it's condition remains true.
- Like an `if` statement but runs continuously until condition is no longer true.

```java
while (cond) {
    // Code to run
}
```

# `do-while`

- A `do-while` loop is very similar to a `while` loop except it guarantees that the associated block of code will run **AT LEAST** once.

```java
// Runs the code AT LEAST once
do {
    // Code to run
} while (cond);
```
