# Format Specifiers

- Defines and formats a type of data to be displayed.
- Use `%` followed by a certain character depending on the type of data to be displayed.
  - `%c` -> Character
  - `%s` -> String
  - `%d` -> Integer
  - `%u` -> Unsigned Integer
  - `%f` -> Float
  - `%lf` -> Double
  - `%lld` -> Long Long Integer
  - `%llu` -> Unsigned Long Long Integer
  - `%.1` -> Decimal Precision
  - `%1` -> Minimum Field Width
  - `%-` -> Left Align
