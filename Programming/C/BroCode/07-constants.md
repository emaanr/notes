# Constants

- A _constant_ is a fixed value that cannot be altered by the program during its execution.
- General declaration format for a constant looks like `const [data type] [VARIABLE NAME] = value;`
    ```c
    float pi = 3.14159;         // Non constant version
    const float PI = 3.14159;   // Constant version
    ```
