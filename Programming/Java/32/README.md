# Definition

- We can pass objects as arguments.
- This can be done by declaring the object you want to pass as a parameter.

```java
class PassedObject {
    // Code
}

class Object {
    // Constructor
    Object(PassedObject obj) {
        // Code
    }
}

class Main {
    Object obj = new Object(PassedObject);
}
```
