# Definition

- Can be used to connect two or more expressions.

# And `&&`

- Both conditions must be true.
- Denoted by `&&`.

```java
if (cond && cond) {
    // Both conditions were true
} else {
    // Both conditions were not true
}
```

# Or `||`

- Either condition must be true.
- Denoted by `||`.

```java
if (cond || cond) {
    // One of the conditions was true
} else {
    // Neither conditions were true
}
```

# Not `!`

- Reverses boolean value of a condition.
- Denoted by `!`.

```java
if (!cond) {
    // Condition was not true (condition ultimately evaluates to false)
} else {
    // Condition was true
}
```
