# `for-each`

- Traversing technique to iterate through the elements in an array/collection, such as `ArrayList`.
- Less steps and more readible, but less flexible.

```java
for (datatype elem : elemList) {
    // Code to run
}
```

- This can be read as:
  - For each `elem` in `elemList`, run the associated block of code.

```java
for (datatype elem : elemList) {
    System.out.println(elem);
}
```

- This would print every `elem` in the `elemList` on its own line.
