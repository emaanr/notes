# Definition

- Statement that allows a variable to be tested for equality against a list of values.
- Can be more eficient than using many `if-else` statements.
  - Unlike `if-else` statements, the `switch` can have a number of possible execution paths.

# Data Types

- Works with:
  - Primitives:
    - `byte`
    - `short`
    - `char`
    - `int`
  - Wrappers:
    - `Character`
    - `Byte`
    - `Short`
    - `Integer`
  - Other:
    - `enum`
    - `String`
