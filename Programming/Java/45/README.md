# `File`

- An abstract representation of file and directory pathnames.
- Exploring some methods of the `File` class in Java.
- `File file = new File(filepath)`.

## `getPath()`

- Returns the path relative to the current working directory.
  - Also know as a "Relative Path".

## `getAbsolutePath()`

- Returns the path relative to the entire computer (disk).
  - Also known as the "Absolute Path".

## `isFile()`

- Returns true if the callee is a file, and false otherwise (meaning its not a file of course).
