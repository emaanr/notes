# `for`

- Executes a block of code some specified, limited number of times.
- A `while` loop could run infinitely, but a `for` loop runs a predetermined number of times.
- A `for` loop typically has three pieces to it:
  1. Pre-Condition
  2. Condition
  3. Post-Condition

```java
for (pre-cond; cond; post-cond) {
    // Code to run
}
```

> Note that in programming, we start counting from 0 rather than 1.
>
> - If we want to run for `10` iterations, that is count 0 to 9 or 1 to 10.

## Pre-Condition

- This is the set up for the Condition and Post-Condition.

## Condition

- While this is true, we will continue to run the block of code.

## Post-Condition

- This is what happens after every iteration.
- Can also be moved into the code block itself, typically at the end of it.
