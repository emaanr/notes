# Definition

- Expression is comprised of operands and operators.

## Operands

- Values,
- Variables,
- Numbers,
- Quanitity.

## Operators

- `+`:
  - Addition.
- `-`:
  - Subtraction.
- `*`:
  - Multiplication.
- `/`
  - Division.
- `%`
  - Modulus (Remainder).

# Augmented Operators

- `++`
  - Increments variable by 1.
- `--`
  - Decrements variable by 1.
- `+=`
  - Increments variable by value after equal sign and assigns the result to the same variable.
- `*=`
  - Multiplies variable by value after equal sign and assigns the result to the same variable.
- `/=`
  - Divides variable by value after equal sign and assigns the result to the same variable.
- `%=`
  - Modulus variable by value after equal sign and assigns the result to the same variable.
